# Project LSAutoAndroid

Workshop management application
Integrated with the application
[LSAuto](https://github.com/DanielTrosko/lsauto)
## Program features
```
* Login
* Displaying currently repaired cars
* Display all cars
* Displaying car details
```


## Built With
